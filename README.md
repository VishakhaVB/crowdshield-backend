# CrowdShield Backend

Backend API for **CrowdShield** – a scalable crowd safety and guidance system designed for managing large-scale public events.

This backend handles authentication, dashboards, and event-related data services.

---

## 🚀 Tech Stack

- Node.js  
- Express.js  
- JavaScript  
- REST API Architecture  

---

## 📂 Project Structure


crowdshield-backend/
│── data/ # Static or seed data
│── models/ # Database models
│── routes/ # API route definitions
│── services/ # Business logic
│── .env.example # Environment variables template
│── index.js # Entry point of application
│── package.json # Project metadata & dependencies
│── verify_backend.js # Backend verification script
│── verify_pilgrim.js # Pilgrim service verification

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/VishakhaVB/crowdshield-backend.git
```

2️⃣ Navigate to project folder
```
cd crowdshield-backend
```

3️⃣ Install dependencies
npm install
4️⃣ Create a .env file

Use .env.example as reference.

5️⃣ Start the server
npm start
